# 英単語テスト v2 仕様書

## 概要

英単語テスト v2 は、Excelファイルから英語テスト用PDFを自動生成するWebアプリケーションです。

---

## 対応問題タイプ

### 1. 単語テスト（Vocabulary）
- **形式**: 英単語 → 日本語意味（または逆方向）
- **シート名**: `vocabulary`, `vocab`, `単語`
- **必須列**: `word`, `meaning`
- **出力**: 2列レイアウト

### 2. 4択問題（Multiple Choice）
- **形式**: 問題文 + 4つの選択肢
- **シート名**: `multiple_choice`, `grammar`, `4択`, `文法`
- **必須列**: `question`, `choice1`, `choice2`, `choice3`, `choice4`, `answer`
- **任意列**: `explanation`（解説）
- **出力**: 1列レイアウト

### 3. 並べ替え問題（Reorder）
- **形式**: 日本語の指示 + 並べ替える語句
- **シート名**: `reorder`, `並べ替え`, `整序`
- **必須列**: `prompt`, `words`, `answer`
- **任意列**: `hint`（最初の語句のヒント）
- **出力**: 1列レイアウト

### 共通オプション列
| 列名 | 説明 |
|------|------|
| `number` | 問題番号 |
| `section` | セクション/単元名 |
| `book` | 教材名 |
| `tag` | 分類タグ |

---

## Excelフォーマット

### 列名エイリアス

システムは以下のエイリアスを自動認識します：

| 標準名 | 認識される別名 |
|--------|---------------|
| `word` | `english`, `英語`, `単語` |
| `meaning` | `japanese`, `日本語`, `意味` |
| `question` | `問題`, `問題文` |
| `choice1-4` | `選択肢1-4`, `1-4` |
| `answer` | `正解`, `解答` |
| `prompt` | `指示`, `日本語`, `問題` |
| `words` | `語句`, `並べ替え` |
| `number` | `no`, `番号`, `#` |
| `section` | `セクション`, `単元`, `lesson` |

### 並べ替え問題の words 形式
- `/` または `,` で区切り
- 例: `I / have / a pen` または `I, have, a pen`

---

## PDF出力デザイン（案C: 余白区切り型）

### デザインコンセプト
- **シンプル**: 装飾を最小限に
- **余白ベース**: 枠線ではなく余白で区切り
- **低インク消費**: 印刷コスト削減

### セクションヘッダー
- 薄いグレーテキスト（#555555）
- 細い横線で区切り

### 問題番号
- 控えめな表示（#666666）
- 右寄せ配置

### 解答表示
- 赤色（red）で強調
- 問題PDFでは下線のみ

## API/ルート

### 主要ルート

| メソッド | パス | 説明 |
|---------|------|------|
| GET | `/` | トップページ |
| GET/POST | `/upload` | ファイルアップロード |
| GET/POST | `/make` | 問題作成（旧版） |
| GET/POST | `/make_v2` | 問題作成（新版・複数タイプ対応） |
| GET | `/download` | PDFダウンロードページ |
| GET | `/download_zip` | ZIP形式でダウンロード |
| GET | `/login` | 認証ページ |
| POST | `/api/analyze` | Excelファイル解析API |

### /make_v2 パラメータ

| パラメータ | 必須 | 説明 |
|-----------|------|------|
| `filename` | ○ | 対象Excelファイル |
| `pattern` | ○ | `mixed`, `vocabulary`, `multiple_choice`, `reorder` |
| `count` | ○ | 出題数 |
| `num_sets` | - | 作成部数（デフォルト1） |
| `vocab_direction` | - | `en-ja` または `ja-en` |
| `sections` | - | 出題セクション（複数選択可） |

## レート制限

### 非認証ユーザー（Public）
| 項目 | 制限値 |
|------|--------|
| 1リクエストあたり最大部数 | 2部 |
| 1日あたり最大部数 | 10部 |
| 1分あたり最大リクエスト | 6回 |

### 認証済みユーザー
| 項目 | 制限値 |
|------|--------|
| 1リクエストあたり最大部数 | 無制限（0=無制限） |
| 1日あたり最大部数 | 無制限 |
| 1分あたり最大リクエスト | 30回 |

### 絶対上限
| 項目 | 制限値 |
|------|--------|
| 1リクエストあたり最大部数 | 100部 |
| 1日あたり最大部数 | 500部 |
| 1分あたり最大リクエスト | 60回 |

## 認証

- Basic認証（オプション）
- 環境変数 `BASIC_AUTH_USER`, `BASIC_AUTH_PASS` で設定
- セッションベースで認証状態を保持

---

## ストレージ

### ローカルモード（デフォルト）
- `uploads/` : アップロードされたExcelファイル
- `uploads/pdfs/` : 生成されたPDFファイル

### R2モード（本番環境）
- 環境変数 `STORAGE_PROVIDER=r2` で有効化
- Cloudflare R2 にファイル保存
- 署名付きURLでダウンロード

## 環境変数

| 変数名 | 説明 | デフォルト |
|--------|------|-----------|
| `FLASK_SECRET_KEY` | セッション暗号化キー | `change-me` |
| `BASIC_AUTH_USER` | Basic認証ユーザー名 | (空) |
| `BASIC_AUTH_PASS` | Basic認証パスワード | (空) |
| `STORAGE_PROVIDER` | `r2` でR2有効化 | (空=ローカル) |
| `PUBLIC_MAX_SETS_PER_REQUEST` | Public 1リクエスト上限 | 2 |
| `PUBLIC_MAX_SETS_PER_DAY` | Public 1日上限 | 10 |
